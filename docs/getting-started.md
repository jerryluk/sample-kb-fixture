# Getting Started

Welcome to the product documentation. This guide helps you get up and running quickly.

## Prerequisites

- Node.js 18+
- An active account

## Installation

```bash
npm install @product/sdk
```

## Quick Start

1. Initialize the SDK with your API key
2. Call `sdk.connect()` to establish a session
3. Start making API calls
