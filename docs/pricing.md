# Pricing

We offer three tiers to fit teams of all sizes.

## Starter — Free

- Up to 5 users
- 1,000 API calls/month
- Community support

## Growth — $49/month

- Up to 25 users
- 50,000 API calls/month
- Email support

## Enterprise — Contact us

- Unlimited users
- Unlimited API calls
- Dedicated support
- Custom SLAs

## FAQ

**Can I switch plans?** Yes, at any time from the billing page.

**Is there a free trial?** The Starter plan is free forever.
